CONTROLS

Please note, this game is intended to be played on gamepad (it can still be played on keyboard if needed though.)

Playstation Buttons are used below

Left Stick/WASD
- movement

X/Space Bar
- jump

□ / J
- light attack
- three button presses in a row will lead to a 3 hit combo

○ / Shift
- roll
- invulnerable for a portion of this animation, so you can use it to dodge through projectiles
- can cancel attacks into roll to follow up with more attacks after launching opponent away


△ / K
- heavy attack
- hold down to charge
- cannot move normally while charging, but can stil jump and roll


Start Button / Esc
- pause/start/restart game
